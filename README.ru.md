# TUI File Manager

Простой консольный файловый менеджер с управлением клавиатурой.  
Написан на C++ с использованием библиотеки **ncurses**.

---

## Возможности

- Навигация по папкам (стрелки вверх/вниз, вход/выход)
- Создание файлов и папок
- Удаление файлов и папок (с подтверждением)
- Управление одной рукой (клавиши в зоне доступа)
- Поддержка UTF‑8 (кириллица в именах)

---

## Установка

### Зависимости

| Дистрибутив | Команда |
|-------------|---------|
| Debian / Ubuntu | `sudo apt install libncurses-dev` |
| Fedora | `sudo dnf install ncurses-devel` |
| Arch / CachyOS | `sudo pacman -S ncurses` |

### Сборка

```bash
g++ fm_beta_v0.2.cc -o filemanager -lncursesw
```

### Запуск

```
./filemanager
```

## Управление

| Клавиша | Действие |
|---|---|
| `↑` / `↓` | Перемещение по списку |
| `→` | Войти в папку |
| `←` | Выйти в родительскую папку |
| `Ins` | Создать новый файл |
| `/` | Создать новую папку |
| `Del` | Удалить выделенный элемент |
| `Ctrl+C` | Выйти из программы |

> Выход по команде `END` пока не реализован. Используйте `Ctrl+C`.
